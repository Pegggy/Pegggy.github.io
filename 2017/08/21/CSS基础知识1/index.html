3.路径

相对路径：

  • css/a.css 当前css文件夹下a.css文件
  • ./css/a.css 与css/a.css相同
  • b.css 当前目录的b.css文件
  • ../imgs/a.png 上层目录的imgs文件夹下的a.png文件
    绝对路径：
  • /Users/hunger/project/css/a.css 本地文件的绝对地址，本地可以找到，但是放到服务器上就不能找到了

5.html和css书写规范

  • 统一使用小写；
  • 不使用内联样式；
  • 块内容缩进；
  • id和class使用有意义的单词，分隔符使用-
  • css属性名冒号后面加一个空格。
